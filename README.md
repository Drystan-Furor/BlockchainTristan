# blockchain-demo

Demo blockchain app by Dave Ligthart.

## Install

`pip install virtualenv`
`pip install pipenv`
`pipenv install`
`pip3 install colorama`

## Run

`pipenv shell`

`FLASK_APP=blockchain flask run`

`open http://127.0.0.1:5000/api/v1/blocks`

## Test

`pipenv shell`

`py.test`

**with output**
`py.test -s`
